Eventos de Teclado

  1. onkeydown


    2. Essa função mostra quando aperta alguma tecla do teclado. Da para utilizar o if para selecionar uma tecla apertada, exemplo comentado no codigo JS. Da pra criar uma situação de bloquear alguma ação com Ex: 'Teclado apertado!' + event.ctrlKey o ctrl ficara como true e as outras teclas false.

  2. onkeyup

    3. Função que mostra quando solta o botão do teclado depois de ter segurado a tecla.

  3. onkeypress

    4. Mostra quando aperta uma tecla
